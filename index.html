자기소개

 목표라는 난관에 부딪혔을 때 멈추지 않고 나아가는 개발자입니다. 웹 솔루션 회사의 경력을 통해 JavaSpring, MySQL, Elasticsearch 등 여러 언어의 경험과 운영 경험을 가지고 있습니다. 문제가 발생할 경우 가능한 한 많은 정보를 수집하고 여러 방면의 시도를 통해 해결 방안을 찾아냅니다.

 운영 중 발생하는 이슈를 로그와 데이터를 기반으로 분석하고, 구조적인 개선을 통해 재발을 방지하는 개발 방식을 중요하게 생각합니다.

사용 기술

Java Spring JSP MySQL Elasticsearch HTML CSS JavaScript
